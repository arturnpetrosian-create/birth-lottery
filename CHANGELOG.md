# Changelog

## v2.0 — 2026-05-01

### Added

- Раздел **§ VI** («Среди всех, кто когда-либо рождался») в статической версии и Streamlit: сравнение выбранной пары страна+год с оценкой **PRB (Kaneda & Haub, 2022)** ~117 млрд рождений; доли для года, кумулятива по стране за 1950–2024, доля ныне живущих относительно 117 млрд; ссылка на первоисточник PRB.
- Модуль `domain/prb_ever_lived.py` с константой `EVER_LIVED_PRB_2022` и вспомогательными функциями форматирования.
- В метаданных JSON: `world_births_sum_1950_2024_persons`; при сборке из xlsx — `world_population_july_2024_thousands`, когда в источнике есть подходящая колонка.
- Тесты `tests/test_data.py` (ряды стран, сумма vs мир ≥95%, бенчмарки RUS/USA/JPN, константа PRB, согласованность суммы рождений).
- GitHub Actions CI: `ruff`, `pytest`, минимальная проверка `static/index.html`.
- Pre-commit: `ruff` (Python), `prettier` для части статики/доков (`static/index.html` исключён из prettier из‑за встроенного JSON).

### Audit / copy / methodology

- Hero и подсказки: годовой объём живорождений описан с учётом изменчивости 1950–2024; «лотерея» явно как **мысленный эксперимент** условной вероятности по когорте года, не как «шанс родиться» у уже существующего человека.
- Контраст вторичного текста: `--ink-light` приведён к **#666663** (AA на фоне бумаги).
- Документация `docs/METHODOLOGY.md`: дисклеймер ООН по границам и источникам; блок **§8 PRB**; перенумерация расширений под **§9** (подпункты 9.1–9.4).
- `README.md`: уточнены формулировки без вводящих в заблуждение обобщений.
- Изотип: убраны тысячи `setTimeout` на точку — переключение классов синхронно в цикле.

### Known limitations (не закрыто в этом релизе)

- CDN (d3, topojson, шрифты) без SRI; при недоступности CDN карта может не загрузиться (см. консоль).
- Цветовая шкала карты не проходила отдельную проверку color-blind симулятором; при необходимости — смена палитры на последовательную «color-blind safe».
- На узких экранах полосы рейтинга топ-стран могут быть упрощены; приоритет — читаемость и скорость.
