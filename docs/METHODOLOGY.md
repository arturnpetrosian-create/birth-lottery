# Методология

## 1. Источник данных

Все оценки числа живорождений по странам взяты из:

> United Nations, Department of Economic and Social Affairs, Population
> Division (2024). **World Population Prospects 2024.** Online Edition.
> https://population.un.org/wpp/

Конкретный файл: `WPP2024_GEN_F01_DEMOGRAPHIC_INDICATORS_COMPACT.xlsx`,
лист `Estimates` (1950–2023) + `Medium variant` (2024 — оценка из medium-сценария).

Используемая колонка — `Births (thousands)`: оценочное число живорождений
(live births) в году. Лицензия — **CC&nbsp;BY&nbsp;3.0&nbsp;IGO** (свободное
использование с указанием источника).

## 2. Что мы считаем

Для года&nbsp;Y и страны&nbsp;C:

```
P(C | Y) = births(C, Y) / world_births(Y)
```

Это **доля живорождений в стране&nbsp;C среди всех живорождений мира за тот же
год&nbsp;Y** — иначе говоря, удельный вес страны в годовой когорте новорождённых.

Содержательная интерпретация: априорная вероятность того, что случайно
выбранный младенец из глобальной когорты родился именно в&nbsp;C.
Это «лотерея места рождения», без поправки на дальнейшую выживаемость
и&nbsp;миграцию.

## 3. Что НЕ учтено

| # | Эффект | Куда уходит вероятность |
|---|--------|-------------------------|
| 1 | **Младенческая и детская смертность** (U5MR — under-five mortality rate). | До 25–30% в развивающихся странах 1950-х. |
| 2 | **Миграция** в первые годы жизни. | Влияет на «где я вырос», но не на «где я родился». |
| 3 | **Изменение государственных границ.** | UN&nbsp;WPP ретроспективно использует современные (на&nbsp;2024) границы. |
| 4 | **Subnational variation** (распределение внутри страны). | Регион/город — не наш масштаб. |

## 4. Современные границы

UN&nbsp;WPP&nbsp;2024 применяет современные границы стран **ретроспективно**.
Это стандартная практика для длинных временных рядов в&nbsp;демографии:
иначе данные были&nbsp;бы не сопоставимы между периодами и&nbsp;пришлось бы
вручную учитывать каждый раздел/слияние.

Конкретные примеры:

- **Россия в&nbsp;1950** = территория современной РФ (≈&nbsp;РСФСР), не СССР.
- **Бангладеш в&nbsp;1960** = территория современной Бангладеш (Восточный Пакистан).
- **Германия в&nbsp;1960** = объединённая территория ФРГ&nbsp;+&nbsp;ГДР.
- **Эритрея в&nbsp;1980** = территория современной Эритреи, отделённая
  от&nbsp;Эфиопии только в&nbsp;1993.
- **Южный Судан в&nbsp;1990** = выделенная из Судана территория, ставшая
  государством в&nbsp;2011.

## 5. Методы оценки в&nbsp;самих данных ООН

UN&nbsp;WPP не публикует «сырых данных», а строит **сглаженные модельные
оценки** через cohort-component projection с&nbsp;байесовским сглаживанием
(Bayesian hierarchical models, особенно для&nbsp;TFR&nbsp;и&nbsp;e₀)
на основе четырёх семейств источников:

1. **Vital registration (VR / гражданская регистрация рождений и&nbsp;смертей)** —
   там, где она полная и&nbsp;точная (в основном развитые страны и&nbsp;часть
   стран Латинской Америки, Восточной Европы, Восточной Азии).
2. **Переписи населения** — около 1&nbsp;910 национальных переписей,
   проведённых в&nbsp;1950–2023 гг.
3. **Демографические выборочные обследования** (DHS, MICS, World Fertility
   Surveys, RHS) — около 3&nbsp;189 национальных выборок.
4. **Косвенные методы** (P/F&nbsp;ratio, reverse&nbsp;survival, own-children,
   cohort-component) — для стран без надёжной vital&nbsp;registration.

Для оценки TFR используется Bayesian hierarchical model
(Alkema&nbsp;et&nbsp;al.&nbsp;2011, Šev&zcaron;íková&nbsp;et&nbsp;al.&nbsp;2016),
для смертности — Lee–Carter и Bayesian Pop projections.

### Точность

Грубый ориентир по&nbsp;типу режима регистрации:

| Группа | Точность ±, % | Пример |
|---|---|---|
| VR полная и проверенная | 1–2 | Германия, Япония, Норвегия |
| VR частичная / переписи + обследования | 5–10 | Бразилия, Индия, Индонезия |
| Indirect&nbsp;estimation, страны затяжного кризиса | 15–20 | Афганистан, Сомали, ДРК, Йемен |

## 6. Покрытие

- **Лет:** 75 (1950–2024).
- **Стран и территорий:** 211. После фильтрации по&nbsp;пиковому числу
  живорождений&nbsp;&lt;&nbsp;1&nbsp;000/год исключены Ватикан, Токелау,
  Питкэрн, Святая&nbsp;Елена и&nbsp;подобные микросущности — оставлять их
  бессмысленно: статистический шум при&nbsp;таких малых&nbsp;n больше сигнала.
- **Country-year наблюдений:** ≈&nbsp;15&nbsp;800.

## 7. Сборка датасета

`scripts/build_data.py` делает следующее:

1. Читает `Estimates` и&nbsp;`Medium variant` из&nbsp;xlsx UN&nbsp;WPP&nbsp;2024.
2. Фильтрует по `Type == "Country/Area"` и&nbsp;по&nbsp;диапазону&nbsp;1950–2024.
3. Вытаскивает мировые тоталы из&nbsp;строк `Region, subregion, country or area * == "World"`.
4. Применяет фильтр по&nbsp;peak births &geq;&nbsp;1&nbsp;000/год.
5. Размечает регион (Africa / Asia / Europe / Americas / Oceania по&nbsp;UN&nbsp;geoscheme).
6. Дописывает русские названия (`RU_NAMES` в&nbsp;скрипте).
7. Сохраняет в&nbsp;компактный JSON: ≈&nbsp;100&nbsp;KB на&nbsp;все страны и&nbsp;годы.

## 8. Возможные расширения

### 8.1 Добавить pre-1950

Для 1900–1949 нет единого качественного источника. Возможные подходы:

- **Maddison Project** — данные о&nbsp;численности населения по&nbsp;странам с&nbsp;XIX&nbsp;в.
- **Caldwell&nbsp;&&nbsp;Schindlmayr (2002)** — оценки CBR (crude&nbsp;birth&nbsp;rate)
  по&nbsp;регионам.
- **Human Fertility Database** (HFD) — точные cohort-данные для&nbsp;~30
  развитых стран с&nbsp;конца XIX&nbsp;в.

Ожидаемая погрешность для&nbsp;большинства стран до&nbsp;1950 — ±20–30%.

### 8.2 Учесть выживаемость

Если хочется ответа на&nbsp;вопрос «вероятность родиться в&nbsp;C&nbsp;и&nbsp;дожить до&nbsp;5&nbsp;лет»:

```
P(survive to 5 | C, Y) = 1 − U5MR(C, Y) / 1000
```

Данные U5MR&nbsp;есть в&nbsp;том же&nbsp;файле `WPP2024_GEN_F01`. Для&nbsp;~95%&nbsp;стран
это снижает «вероятность вырасти здесь» в&nbsp;`(1 − U5MR/1000)`&nbsp;раз,
что в&nbsp;развивающихся странах&nbsp;1950-х&nbsp;могло достигать 25–30%.

### 8.3 Добавить TFR&nbsp;/&nbsp;CBR&nbsp;/&nbsp;e₀

В&nbsp;том&nbsp;же датасете есть:

- **TFR** (Total Fertility Rate, среднее число рождений на&nbsp;женщину).
- **CBR** (Crude Birth Rate, число рождений на&nbsp;1&nbsp;000 населения).
- **e₀** (life&nbsp;expectancy at&nbsp;birth, ожидаемая продолжительность жизни).

Их добавление позволит контекстуализировать «лотерею» — например, отделить
страны в&nbsp;разгаре демографического перехода (TFR&nbsp;5–7) от&nbsp;стран
post-transitional&nbsp;fertility (TFR&nbsp;&lt;&nbsp;2.1).

### 8.4 Когортные траектории

Сейчас атлас работает в&nbsp;«календарной» логике (по&nbsp;годам календаря).
Альтернатива — следить за&nbsp;**когортой** на&nbsp;протяжении всей её жизни:
рождённые в&nbsp;1992 году в&nbsp;РФ&nbsp;→ численность 0&nbsp;лет, 1&nbsp;год, …,
33&nbsp;года&nbsp;на&nbsp;2025. Это требует подключения смертности по&nbsp;возрастам и&nbsp;миграционных потоков.
